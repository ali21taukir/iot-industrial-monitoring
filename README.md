# iot-industrial-monitoring
Designed and developed an IoT-based system for remote monitoring and operation of an industrial plant, enabling real-time data acquisition, control through a centralized interface, and improved operational safety and efficiency.
# IoT Industrial Monitoring System

Real-time remote monitoring system for industrial & mining environments.
Built at CSIR-CIMFR, Dhanbad.

## Tech Stack
- ESP32/ESP8266 (firmware in C/C++)
- Python (backend services & automation)
- MQTT protocol (device-to-cloud)
- Blynk Cloud (dashboards)
- REST APIs (data integration)

## Features
- Real-time sensor data collection
- Cloud dashboard with threshold alerts
- Modular, scalable architecture
